# captcha
Simple PHP Captcha
# Parameters
<b>mode</b>: <i>"raw"/"b64"</i><br>
<b>length</b>: <i>The length of the generated random number. Accepted values 1-20.</i><br>
<b>type</b>: <i>"png", "jpeg", "gif"</i><br>
<b>tColor</b>: <i>Text colour in hex format without #.</i>
<b>bColor</b>: <i>Background colour in hex format without #.</i>
<b>lColor</b>: <i>Line colour in hex format without #.</i>
